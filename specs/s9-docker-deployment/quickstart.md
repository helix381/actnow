# S9 Quickstart

1. Copy `.env.example` to `.env`.
2. Fill local text proxy values or use a mock value if the text path is not under test.
3. Run `docker compose up --build`.
4. Run database migration.
5. Run seed command.
6. Open the frontend URL.
7. Check API health.
8. Create a test generation task and confirm the worker consumes it.
9. Upload a small test file and confirm it lands in MinIO.
