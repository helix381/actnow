# S7 Quickstart

1. Seed a project with 1 episode, 2 scenes and at least 8 shots.
2. Open the workspace canvas.
3. Send: `把第 8 镜改得更压迫一点，只重生成这一镜`.
4. Confirm the target is Shot 8 in the confirmation card.
5. Approve execution.
6. Check that Shot 8 content and generation status changed.
7. Check the Agent event log contains the full decision trail.
