## 产品定位

**ActNow**（暂名）—— 以 Harness 工程哲学驱动的多路线 AIGC 内容创作平台，面向有质量追求的进阶个人创作者。模型是底层资源，产品价值在于围绕它构建的编排层：**Agent 聊天框贯穿全程，是工作流状态机、上下文管理、多 Agent 调度的用户侧体现**，纯靠对话可驾驭完整创作工作流，画布手动是老手高效路径，二者功能对等。两个产品范式决策：**分水岭交互**（创意阶段=聊天室发散收敛，剧本确定后切换为制作阶段确定性流程，PC=无限画布 / 移动=步骤式，匹配两种创作心智，多数竞品用一套 UI 硬撑全程）+ **双路径对等**（对话与画布功能等价）。生成后端做成可切换是务实约束而非创新点（当前缺便宜 API，产品出提示词、用户外部跑，后续接固定 API；顺带支持切换模型）。MVP 先落地漫剧/短剧路线，架构按多路线设计、将来扩展不重构。

## MVP 范围（漫剧/短剧路线 · 完整一条龙）

### 做

**核心工作流（全做）**
- 入口 A：上传整季剧本 + 创作设置（风格/比例/时长/模型/Skill）
- 入口 B：灵感→大纲→编剧（聊天式发散收敛，生成无版权剧本）
- 剧本拆分（后台自动 → 画布铺开校对）
- 资产提取与管理（角色：人/非人·多形态/年龄/身份；场景；道具：特效类/实物类）
- 全景场景管理 + 3D 导播台（**能力必做、按需可选**；可只用 2D）
- 分镜脚本生成
- 分镜图（图生路径关键帧，整集预览校对）
- 视频片段（文生/图生镜头级可选；语音视频模型直出；图生首尾 1s 变速）
- 合成导出（音效/BGM/音轨编辑）
- 降门槛辅助（预设/模板；入门用户可走通）
- 多路线架构骨架（创意短片/电商入口可见，功能不落地）
- 生成后端可插拔（人在环 / 真实 API / 预置样例）

**规模假设**：MVP = 一集完整短片（数十镜头级）；架构支持整季多集（demo 不必全跑）。

### 不做

- 自研底层图像/视频大模型
- 纯小白傻瓜工具（小白靠预设/模板/教程覆盖）
- 一键做同款（漫剧不做；仅创意视频路线，Roadmap）
- 移动端适配（Roadmap 垫底）
- 发行投流（Roadmap）
- 完整 Skill 系统与市场（Roadmap）

## 交互范式（已定）

**三派融合**：创意聊天室（堆友/OiiOii 风）+ 无限画布（LibTV/Lovart 风）+ 步骤式（LuxReal/巨日禄 风）——取长补短，按阶段分工：

- **创意阶段（灵感→大纲→编剧）= 聊天室**：AI 启发式提问 + 推荐选项，发散→收敛；步骤流水线在此段效果差，实践教训。
- **剧本确定 = 分水岭**，之后是确定流程。
- **制作阶段**：PC = 无限画布节点（效率最高；成功案例可存为工作流模板）；移动 = 步骤式（Roadmap 垫底）。
- **Agent 聊天三态**贯穿全程，与画布手动**功能对等**——新手用嘴、老手上手、可混用。

## MVP 验收标准

| 指标 | 目标 |
|------|------|
| 演示核心场景 | 一句灵感/剧本 → 角色一致性分镜，可现场演示 |
| 端到端跑通 | 剧本→成片全流程贯通（生成可外置回填） |
| 角色一致性 | 同角色不同镜头目测无明显漂移 |
| 全景正反打 | 同场景正反打背景不穿帮 |
| 对话式修改 | 单点修改仅目标镜头重生成，不牵连其他 |
| 后端可切换 | 人在环/真实 API/样例三模式接口预留可切 |
| 断点续作 | 关闭后重进，进度/资产完整恢复 |
| 降门槛可用 | 用模板能走通，入门用户可产出可用初稿 |
| 多路线入口可见 | 创意短片/电商入口可见、引导清晰 |
