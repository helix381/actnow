# 页面与流程

> 交互范式（详见 01 产品形态）：**创意阶段=聊天室**，**剧本确定=分水岭**，**制作阶段 PC=无限画布节点 / 移动=步骤式**。
> **Agent 聊天=核心交互**：可纯对话完成全部编辑动作；画布手动是老手高效路径，二者功能对等、可混用。
> 制作阶段的「剧本拆分/资产/分镜/分镜图/视频/合成」在 PC 上是**画布节点**，移动端**拍平成步骤 tab**——下表「阶段·形态」列标明。
> **顶层阶段骨架 = 剧本 / 资产 / 制作 / 预览 4 步**（"制作"内在画布上展开为 分镜→分镜图→视频 等节点；与 LuxReal 4 步同量级，不强行拆成 5 步 tab）。
> **画布上的资产库**：画布铺不下资产库，入口候选——(a) 聊天面板对侧的**资产库浮窗** / (b) 底部**工具栏入口**（**待定**，见 05）；画布上素材可点击**收藏入库**。

## 主流程

### 流程一：总览（两入口 → 分水岭 → 制作）

```mermaid
flowchart TD
    Start[新建项目·选路线] --> Ent{有剧本?}
    Ent -->|有| A[入口A: 上传整季剧本 + 创作设置]
    Ent -->|没有/避版权| B[入口B: 创意聊天 灵感→大纲→编剧]
    A --> WS[【分水岭】剧本确定]
    B --> WS
    WS --> Make[制作: 拆分→资产→分镜→视频→合成→导出]
    Make -.Roadmap.-> Pub[发行投流]
    Discover[首页·发现瀑布流] -.浏览找灵感.-> B
```
> 注：漫剧 MVP **不做一键同款**（抄同款≠精品）；漫剧的"同款"本质是二创（相同灵感/叙事节奏/设定），属 Roadmap，故不入流程一。一键同款仅创意视频路线（见流程五）。

### 流程二：创意阶段（编剧之前 · 聊天室，发散→收敛）

```mermaid
flowchart TD
    I[一句灵感/模糊想法] --> Q[AI 启发式提问 + 推荐选项]
    Q --> Concrete[灵感由模糊 → 具象]
    Concrete --> Outline[生成大纲]
    Outline -->|对话式调整| Outline
    Outline --> Write[编剧成剧本]
    Write -->|对话式调整| Write
    Write --> Done[剧本确定 →【分水岭】]
    Up[入口A: 上传整季剧本] --> Done
```

### 流程三：制作阶段（编剧之后 · 画布/步骤，确定流程）

```mermaid
flowchart TD
    Done[剧本确定] --> Split[剧本拆分: 后台自动拆 → 画布铺开校对]
    Split --> Asset[拆资产: 角色 人/非人·多形态 / 场景 / 道具]
    Asset -.按需,过场可跳过.-> Pano[720°全景 → 3D 导播台 可选]
    Asset --> SB[分镜脚本: 情绪/运镜/景别]
    Pano -.设机位.-> SB
    SB --> Mode{镜头生成模式}
    Mode -->|图生| Img[分镜图/关键帧: 角色+空间一致]
    Mode -->|文生| Vid
    Img --> Vid[视频片段: 语音随模型直出, 图生首尾1s变速]
    Vid --> Comp[合成 + 音效 BGM/音轨 + 导出 mp4]
    Comp --> End[完成]
    DualPath[["每节点均可: Agent对话改 ∥ 画布手动改"]] -.贯穿.-> Asset
```

**制作阶段子步骤详图**（编剧之后·确定流程，每子阶段一张细图）

#### 流程三-② 拆资产

```mermaid
flowchart TD
    S2[剧本已确定] --> Ext[AI 提取 角色/场景/道具 清单]
    Ext --> Cha[角色]
    Ext --> Sce[场景]
    Ext --> Pr[道具]
    Cha --> CT{人类 or 非人?}
    CT -->|人类| CH[设外貌/性格描述]
    CT -->|非人 动物/怪物/拟人| CN[设物种/形态特征]
    CH --> Form[定多形态: 年龄 少年/成年/老年 · 身份 平民/将军…]
    CN --> Form
    Form --> Base[生成基准图 三视图/锁脸 或 上传参考图]
    Base --> Role[设主配角]
    Sce --> Sce2D[生成 2D 场景资产图 默认]
    Sce2D --> SN{需多机位/正反打?}
    SN -->|是| Pano[升级 720°全景 → 3D 导播台]
    SN -->|否 过场| Keep2D[保持 2D]
```

#### 流程三-③ 分镜脚本

```mermaid
flowchart TD
    S3[资产就绪] --> Gen[AI 按叙事单元/场景生成分镜]
    Gen --> F[每镜: 景别/机位/运镜/情绪/台词/时长]
    F --> Rhythm[识别情绪节奏 → 规划镜头密度/特写]
    Rhythm --> M{含多机位对话/正反打?}
    M -->|是 且场景有全景| Cam[3D 导播台基于全景设机位]
    M -->|否| NoCam[沿用默认机位]
    Cam --> Rev[校对]
    NoCam --> Rev
    Rev --> E{需改?}
    E -->|改字段/增删/调序/重生单镜| Gen
    E -->|通过| Done3[分镜脚本定稿]
```
> **切镜/运镜由内置提示词规则驱动**（AI 按规则规划，用户可改）；外部运镜预设库见 Roadmap。

#### 流程三-③/④ 镜头生成模式分支

```mermaid
flowchart TD
    D[分镜定稿] --> Each[逐镜判断生成模式]
    Each --> Mode{文生 or 图生? 可手动覆盖}
    Mode -->|图生 需关键帧预览/更可控| ToImg[→ ④ 分镜图]
    Mode -->|文生 更快/无启停慢速| ToVid[→ ⑤ 视频, 跳过 ④]
```

#### 流程三-④ 分镜图（关键帧）— 仅图生镜头

```mermaid
flowchart TD

    Asm --> Batch[批量生成分镜图 关键帧]
    Batch --> Prev[整集关键帧预览]
    Prev --> Ck{满意?}
    Ck -->|否| Fix[单张重生 / 对话式改 / 选预设 / 换形态]
    Fix --> Batch
    Ck -->|是| Done4[关键帧定稿 → ⑤ 视频]
```

#### 流程三-⑤ 视频片段

```mermaid
flowchart TD
    S5[镜头待生成] --> Src{来源}
    Src -->|图生| FI[基于分镜图 关键帧 生视频]
```

#### 流程三-⑥ 合成与导出

```mermaid
flowchart TD
    S6[片段就绪] --> Seq[按分镜顺序拼接]
    Seq --> Tr[加转场]
    Tr --> Snd[音效编辑: BGM + 音轨添加/编辑]

    Ok -->|否| Bk[回退改某镜/某轨]
```


## 关键对象与状态

> 定型高度：仅列核心对象 + 一句话关系；**不写字段/完整状态机/接口**（完整数据模型与状态机留 Make-PRD，见 05 Q13）。

| 对象 | 一句话关系 | 状态 |
|------|-----------|------|
| **Project（项目）** | 一次创作的根，属于某 Route | ✅ 已确认 |
| **Route（路线）** | 漫剧/创意短片/电商，决定流程与题材；Project 选一个 | ✅ 已确认（漫剧 MVP，余 Roadmap） |
| **Episode（集）** | Project 下分集（整季多集），含叙事单元/场景/镜头 | ✅ 关系确认；分集粒度待 PRD |
| **Scene（场景）** | 剧情发生地，可仅 2D 图或升级 720° 全景；被多个 Shot 复用 | ✅ 已确认 |
| **Shot（镜头）** | 最小制作单元，属某 Scene/Episode | ✅ 关系确认；字段待 PRD |
| **Character（角色）** | 人/非人，含多个 CharacterForm | ✅ 已确认 |
| **CharacterForm（角色形态）** | 同角色的年龄/身份变体，各有基准图，被 Shot 引用以保一致 | ✅ 概念确认；引用机制待 PRD |
| **Prop（道具）** | 特效类/实物类，归属场景/角色 | ✅ 已确认 |
| **Asset（资产）** | Character/Scene/Prop/基准图/全景 的统称；项目内 + 全局库可跨项目复用 | ✅ 两层确认；复用机制待 PRD |
| **GenerationTask（生成任务）** | 挂在某 Shot/Asset 上，走生成适配层 | ✅ 存在确认；完整状态机待 PRD |
| **GeneratedFile（生成产物）** | 图/视频/成片，关联回 Shot/Asset/Project | ✅ 关系确认；存储方案待 PRD |
| **WorkflowTemplate（工作流模板）** | 成功画布案例/参数沉淀，可复用 | ⏳ 概念确认；Roadmap 细化 |

> 另有 Storyboard（分镜脚本，Episode→逐 Shot）、Keyframe（分镜图，仅图生 Shot）作为派生对象——关系已确认，细节待 PRD。

## 高风险流程识别

> 仅识别风险节点与性质，**不给解法/状态机**（解法留 Make-PRD，未决项见 05）。

**1. 异步生成（排队/等待/失败/重试/部分完成）**
- 节点：角色基准图、全景图、分镜图（批量）、**视频片段（最久）**。
