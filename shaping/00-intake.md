## 用户原始想法

做一个**多创作路线的 AI 视频创作平台**，对标 **LuxReal**——「不自研底层模型，也要做出接近 LuxReal 的效果」，界面风格也想往 LuxReal 靠。平台从骨架上就是多路线（漫剧/短剧、创意设计、电商视频等并列），**MVP 先落地漫剧/短剧路线**，其余路线在导航上保留入口、底层架构按多路线设计，将来扩展不重构。

核心价值「让一个人顶一个团队」——把编剧、分镜、角色设计、剪辑的多角色协作流程，压缩为单人可独立完成的一条龙 AI 工作台，且每个环节都能介入精控。

**目标用户**：主打**有付费能力、有质量追求、想要精细画面控制的进阶个人创作者**；入门用户不是核心，但通过预设、模板、一键做同款，以及教程/课程科普来覆盖。非「面向纯小白的傻瓜工具」。

**生态与商业化愿景**：像 LuxReal 一样提供 Skill 组件系统；进阶用户（大佬）的 Skill 组件可付费给入门用户使用，形成创作者付费经济；配套教程/课程。（这些属愿景/二期，MVP 先用预设/模板/一键同款做轻量雏形。）

**创作者本人背景**：当前是提示词工程师，正转 AI 产品经理。这个 demo 一举三用——求职作品集、为「一人公司」创业验证方向、练手体现产品 sense 与提示词工程能力。核心演示诉求：从一句剧本 → 生成一段带角色一致性的分镜，并能走完整一条龙到成片。

**关键约束（决定产品形态）**：
- 不自研底层图像/视频大模型，以调 API + 开源项目为主。
- 生成后端可插拔，架构**预留真实生成 API 接口**。开发/测试期不接真 API：产品内导出提示词→拿到外部（公司报销的工具）跑→结果回传产品（人在环）；正式演示时接上真实 API 切换即可。完整流程必须做扎实，视频环节允许外部做完上传回填。
- 仅桌面 Web，不做移动适配。
- 时间：越快越好。
