## 时长驱动分组

- 每组总时长 <= 15秒
- 每组分镜数量 <= 8个
- 单条分镜时长 >= 1.0秒
- 含对白：每10个字约2秒
- 纯动作/空镜：2-4秒

## 场景边界硬隔离

同一组内不允许出现跨场景的条目。新场景必须出现在下一组第一条。

## 转场

默认硬切。仅以下情况允许其他转场：
- 全剧终 → 淡出/黑场
- 角色晕厥/意识消失 → 黫场
- 画面明确包含"切黑" → 黑场

## GROUP 格式

每组以头行开始：
GROUP_N | 场景名 | 场景ID | 总时长
[规划：...]（分组理由+景别策略+关键运镜理由）
然后是该组所有分镜行。
