## 景别7选一

特写 / 近景 / 中景 / 中全景 / 全景 / 远景 / 大远景

## 前置校验

扫描画面内容，计数独立视觉元素：
- 每个角色算1个
- 每个需在画面中可见的物件/环境细节算1个
- 涉及全身动作弧额外+1

规则：
- 元素>=3 或全身位移 → 强制中景以上，禁近景/特写
- 2+角色同时出现且均需可辨认 → 强制中景以上
- 仅元素<=2 且单角色时 → 允许近景/特写

## 情绪→景别+运镜映射

| 情绪 | 景别 | 运镜 |
|------|------|------|
| 愤怒积累 | 近景 | 固定 |
| 愤怒爆发 | 特写 | 固定（禁缓推） |
| 被压制/恐惧 | — | 仰拍30-45° |
| 施压方 | — | 俯拍30-45° |
| 孤独/绝望 | 远景 | 固定（禁跟随） |
| 紧张/悬念 | 中景→近景止步 | 缓推（禁横移） |
| 爽点爆发 | 仰拍30-60° | 快推 |
| 情感突破 | 近景→特写 | 极缓推（禁快推） |

## 起承转合

- 起段（第一条）：全景/中全景/中景，时长2-4s
- 承段：景别渐进不可逆，同一景别连续不超2个
- 转段（倒数第一条，有戏剧变化）：景别跳跃，总时长<=3s
- 合段（倒数第一条，收束）：固定镜头，景别回退一级
