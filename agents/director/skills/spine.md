## 认知防火墙

ep < crisis_ep 的任何集：a / res / wc 禁止出现终极真相词汇；主角推演100%指向 red_herring。

## 钩子系统（6类型 + 3强度）

### 6种合法钩子类型

| 类型 | 说明 | 适用位置 |
|------|------|---------|
| 动作前夜型 | 大战一触即发 | 日常集/中段 |
| 金手指悬念型 | 能力新形态即将解锁 | 金手指升级集 |
| 身份揭露型 | 真实身份/关系曝光 | paywall/高潮前 |
| 情感临界型 | 关系到断裂边缘 | 情感线关键集 |
| 信息炸弹型 | 炸弹级信息抛出 | midpoint/crisis |
| 危机爆发型 | 物理危机降临 | 通用 |

### 3档强度

- ★★★★★（五星）：身份揭露型/信息炸弹型 → paywall_ep / midpoint / climax前一集
- ★★★★（四星）：金手指悬念型/情感临界型 → 第一幕末/第二幕关键节点
- ★★★（三星）：动作前夜型/危机爆发型 → 第一幕中段/日常集

铁律：paywall_ep-1 必须四星以上，paywall_ep 必须五星。连续2集同强度时第3集必须变。

## 爽点铁律

```
每集    >=1个 C级（有戏剧动作的受挫或小胜，禁空集）
每5集   >=1个 B级（阶段BOSS击败 / 金手指大升级 / 情感重要转折）
每10集  >=1个 A级（重大反转 / 全场震惊 / 情感确认）
midpoint   强制 A级
paywall_ep A级苗头+强钩子
crisis     A级（伪高潮崩溃）
climax     A级
```

禁止：连续2集爽点等级相同；ep1爽点超过ep5；金手指ep1全力爆发。

## 成瘾机制（逐集应用）

- **W1 50%确定性**：每个爽点有意外细节>=1，观众猜对一半猜不到另一半
- **W2 损失框架**：集尾用"失去/死亡/被夺走"框架，非"即将获得"
- **W3 蔡格尼克**：每关闭1个悬念->打开>=2个新悬念
- **W4 好奇心缺口**：前3秒有信息缺口；集尾保留>=1个未解缺口
- **W5 寄生情感绑定**：ep1前30秒建立第一人称感；ep3前角色死亡观众会痛
- **W6 沉没成本**：>=2条独特规则让观众理解代价积累；三集有可见进度递进
- **W7 近似失败钩子**：每集>=1次"差一点成功"结构

## 逐集骨架字段

严格 N 条，不省略不跳号。每集：

`ep / sf / arc / cast / a（电报体行动） / res（不可逆后果） / cst（本集代价） / hk（集尾悬念<=15字） / hook_type / hook_strength / prm / dopamine（C/B/A） / wc（场景物理状态）`

**EP1 专项铁律**：
- a 第一句用公式A或公式B（见 ep123 skill）
- 金手指 ep1 出现预告，<=ep3 完整觉醒
- ep1.hk 必须 A 级强度（身份核弹/隐藏Boss现身/金手指临界）
- ep2 开场是 ep1 末帧直接因果（零断层）
- climax 必须与 ep1 形成场景/台词/画面镜像
