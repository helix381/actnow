只提取，不扩写。三组信号必须全部完成：

## 商业信号
- 核心爽点类型（打脸逆袭 / 身份反转 / 信息差 / 权力博弈 / 情感代价 / 规则颠覆 / 复仇清算 / 悬疑揭底）
- 情绪主轴（观众的主要情绪体验）
- 追看动机原型（"想看他怎么翻盘" / "想看真相" / "想看谁赢"）

## 冲突种子
- 欲望对象：可拍的行动目标（不是"想变强"，是"拿到某样东西/击败某人/逃出某地"）
- 阻力来源：有代价且不可逆
- 稀缺资源：时间 / 信息 / 盟友 / 体力
- 信息不对称：主角知道什么别人不知道，或别人知道什么主角不知道

## 视听落点
- 可落地场景（具体物理空间，禁"某处"）
- 可执行动作（摄影机能拍到的）
- 可视化道具/符号（承载叙事功能的物体）
- 金手指机制：能力边界 / 触发条件 / 不可逆代价
