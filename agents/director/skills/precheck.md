## 合规一票否决

命中任一条 → 立即终止，不进入后续流程：

| 类别 | 条件 | 处理 |
|------|------|------|
| 政治敏感 | 涉及现实政治人物/事件影射 | 直接拒绝 |
| 宗教敏感 | 丑化或歪曲宗教内容 | 直接拒绝 |
| 历史翻案 | 对已定性历史事件/人物翻案 | 直接拒绝 |
| 暴力血腥 | 超出平台审核尺度 | 直接拒绝 |
| 儿童危险 | 儿童暴力/性暗示/未成年人危险场景 | 直接拒绝 |
| 知名IP | 四大名著/革命历史人物直接入主线 | 替换为原创架空设定 |

高门槛预警（不拒绝，需确认制作条件）：都市情感/古装历史/军事题材/涉案题材 → 需100万以上投资。

## 题材赛道判断

- **红海**：榜单高频、竞争激烈，需差异化切入（如：赘婿/战神/重生/甜宠）
- **蓝海**：榜单低频或新兴、供给不足（如：末世囤货/系统流/职场逆袭）
- **死亡区**：平台调性不匹配或用户无付费意愿

判断依据：META-002 题材赛道参考（references/genre_tracks.md）

## 集数决议

写入 meta.nodes：

| policy | N 的取值 |
|--------|---------|
| CUSTOM_EXACT | episode_count_custom |
| RANGE | 区间中位整数 |
| AUTO | 短剧 15-30 集；中剧 31-60 集（按题材密度判断） |

节点坐标：
```
paywall_ep : N<=20 -> max(3, round(N*0.35)); N=21~40 -> 10; N>=41 -> round(N*0.20)
midpoint   : round(N*0.50)
crisis     : floor(N*0.85)
climax     : N
```

## 画风读取

按 meta.style 决定角色外形描述词规范：

| style | 规范 |
|-------|------|
| realistic | 五官结构词（颧骨/眼距/肤色/发型结构），禁"帅气/美丽/温柔" |
| 2d_korean | 漫画夸张词（眼型/脸型/头身比），禁写实皮肤描述 |
| 3d_animation | 材质渲染词（光泽度/骨骼线条/特效元素） |
