## desc 一字不改

assets.characters[].vis 是角色外形的唯一来源。填入提示词时不删不改不润色。

## 提示词结构

每个角色提示词 = desc原文 + 状态词 + 姿态词 + 视线 + 表情AU + 背景要求

- **状态词**（三选一）：服装动态 / 能量特效 / 气质标签
- **姿态词**：站/坐/跪/转身/回头等
- **视线**：正视镜头/侧视/俯视/仰视
- **表情AU**：FACS编码（AU12嘴角上扬=微笑，AU4皱眉=愤怒，等）

## 配角精简

配角行：desc核心词3-5个 + 位置 + 状态词1词 + 视线方向

## 背景要求

- 角色设定图：正面半身，稳定造型，干净灰色/白色背景
- 3人及以上群像：每个角色独立表情，不可共用
