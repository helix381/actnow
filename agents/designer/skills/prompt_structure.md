## 五板块黄金公式

好的提示词 = 5个板块有序排列，每个信息只出现一次。

1. **主体（前置）**：用可量化的视觉动作替代情绪词
2. **光线（中段）**：光源方向、光比、色温
3. **镜头**：镜头类型+焦段+光圈+拍摄距离+相机角度
4. **细节（末尾）**：色彩饱和度、光线柔和度、背景模糊程度
5. **情绪（末尾）**：可量化的视觉参数

## 语序原则

- 主体词前置 → 光影词中段 → 抽象/情绪词末尾
- 不要在尾部追加，语序错乱导致画面越来越假
- 每个信息只出现一次，重复词分散AI权重

## 剪词三法则

1. 剪掉情绪词 → 用具体参数替代（"电影感"≠有效控制信息）
2. 剪掉重复词 → 相似词不会叠加效果，只分散权重
3. 剪掉人类句式 → 把完整句拆成元素列表，动作之间要有主次
